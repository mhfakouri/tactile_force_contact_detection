# Project Summary

**Title:** Tactile/Force Signal Filtering and Contact Event Detection

**Purpose:** Demonstrate signal-processing skills for robotic manipulation and tactile/force sensing.

**Implemented:**
- Synthetic force/torque-like data generation
- Noisy sensor signal visualization
- Moving-average filtering
- Butterworth low-pass filtering
- Adaptive threshold-based contact onset/release detection
- Contact feature extraction
- CSV and plot export

**Main result:** The default signal contains one contact event. The pipeline detects contact onset at approximately **3.23 s** and release at approximately **7.07 s**, then extracts peak force, mean contact force, duration, impulse, and rise time.

**CV line:** Developed a Python signal-processing pipeline for force/torque-like tactile data, including Butterworth filtering, contact onset detection, feature extraction, and visualization for sensor-based robotic manipulation.
